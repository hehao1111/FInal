## Project Structure
This project consists of two main components:
1. Personal Portfolio Website - A React-based web application
2. Database Management System - Adminer for database management

## Technologies Used
- Frontend: React.js
- Database: PostgreSQL
- Database GUI: Adminer
- Containerization: Docker & Docker Compose
- CI/CD: GitHub Actions

## How to Run
1. Clone the repository
2. Run `docker-compose up`
3. Access the applications at:
   - Personal Website: http://localhost:3000
   - Database GUI: http://localhost:8080

## Development
The project uses Docker Compose for orchestration and includes:
- Web application container
- PostgreSQL database container
- Adminer database management container
